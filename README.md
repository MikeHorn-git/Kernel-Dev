# Usage

```Makefile
Usage: make <target>
Targets:
  help         Display this help message
  build        Run the build script
  deploy       Download, extract, and configure kernel if not present
  module       Build kernel modules in ./src/module
  share        Run QEMU with disk.img and mount shared folder
  clean        Remove built kernel and module files
  ```
